# 安装

  1. node.js安装

    官网下载 mac版安装即可

  2. Electron安装

    sudo npm install electron -g --unsafe-perm=true --allow-root

    npm命令前都要加sudo, 不然会权限不足.

  3. 官方的例子

    git clone https://github.com/electron/electron-quick-start
cd electron-quick-start
npm install
npm start
